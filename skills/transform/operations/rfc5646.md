# RFC5646 Transform

**Reference:** [RFC5646 | SailPoint Developer Community](https://developer.sailpoint.com/docs/extensibility/transforms/operations/rfc-5646)

## Overview

Use the **RFC5646** transform to convert an incoming string into an RFC 5646 language tag. Accepts:

- Three-letter language abbreviation (e.g. `SPA`)
- Language name (e.g. `Spanish`)

**Output:** RFC5646 language tag (e.g. `es`).

**Other considerations:** If the input is not a valid language code or name, the transform returns null. See the [SailPoint docs](https://developer.sailpoint.com/docs/extensibility/transforms/operations/rfc-5646) for the full conversion table.

## Transform structure

```json
{
  "type": "rfc5646",
  "name": "RFC5646 Transform"
}
```

## Top-level properties

| Property                  | Type    | Required | Description                                                                                                                  |
| ------------------------- | ------- | -------- | ---------------------------------------------------------------------------------------------------------------------------- |
| `type`                    | string  | Yes      | Must be set to `rfc5646`.                                                                                                    |
| `name`                    | string  | Yes      | The name of the transform as it appears in the UI dropdown menus.                                                            |
| `requiresPeriodicRefresh` | boolean | No       | Whether the transform logic should be reevaluated every evening as part of the identity refresh process. Default is `false`. |

## Attributes

| Attribute | Type   | Required | Description                                                            |
| --------- | ------ | -------- | ---------------------------------------------------------------------- |
| `input`   | object | No       | Explicit input. If not provided, uses input from the UI configuration. |
