# ISO3166 Transform

**Reference:** [ISO3166 | SailPoint Developer Community](https://developer.sailpoint.com/docs/extensibility/transforms/operations/iso-3166)

## Overview

Use the **ISO3166** transform to convert an incoming string into an ISO 3166 country code value. Accepts:
- Alpha2 code (e.g. "ES")
- Alpha3 code (e.g. "ESP")
- Numeric code (e.g. 724)
- English country name (e.g. "Spain")
- Native country name (e.g. "España")

**Other considerations:** If the input is not a valid country code or name, the transform returns null.

## Transform structure

```json
{
  "type": "iso3166",
  "name": "ISO3166 Transform"
}
```

## Top-level properties

| Property                  | Type    | Required | Description                                                                                                                  |
| ------------------------- | ------- | -------- | ---------------------------------------------------------------------------------------------------------------------------- |
| `type`                    | string  | Yes      | Must be set to `iso3166`.                                                                                                    |
| `name`                    | string  | Yes      | The name of the transform as it appears in the UI dropdown menus.                                                            |
| `requiresPeriodicRefresh` | boolean | No       | Whether the transform logic should be reevaluated every evening as part of the identity refresh process. Default is `false`.  |

## Attributes

| Attribute | Type   | Required | Description                                                                                                                       |
| --------- | ------ | -------- | --------------------------------------------------------------------------------------------------------------------------------- |
| `format`  | string | No       | Output format: `alpha2` (default, e.g. "US"), `alpha3` (e.g. "USA"), `numeric` (e.g. "840").                                       |
| `input`   | object | No       | Explicit input. If not provided, uses input from the UI configuration.                                                            |

## Examples

### Example 1: Default alpha2 output

**Input:** `"United States of America"`  
**Output:** `"US"`

```json
{
  "type": "iso3166",
  "name": "ISO3166 Transform"
}
```

### Example 2: Numeric format

**Input:** `"ES"`  
**Output:** `"724"`

```json
{
  "attributes": {
    "format": "numeric"
  },
  "type": "iso3166",
  "name": "ISO3166 Transform"
}
```
